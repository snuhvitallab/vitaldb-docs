## Connection Steps
1. Attach a **Null Modem (F/F)** adapter to the serial port.
2. Connect a direct serial cable from the adapter to the PC via USB-Serial converter.## Device Configuration
1. Navigate to **System Settings → System Menu**. Enter password **`0000`**.2. Select **Interface Configuration**.3. For the connected port (COM1 or COM2), set **Protocol → MEDIBUS** and **Baud Rate → 9600**.
